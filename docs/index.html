Informações adicionais

❥ Como executar o jogo

↬ Instalar o VS Code

↬ Necessário ter o Python 3 instalado no computador

↬ Baixar o zip do jogo (clique no saiba mais)

↬ Para instalar a biblioteca, digite no terminal do VS Code 'pip install pygame'

❥ Como jogar

↬ Use 'aswd' para movimentar Draco Malfoy

↬ Use as setas para movimentar Harry Potter

↬ Colete as horcrux para ganhar pontos e conseguir escapar da sala!

↬ O objetivo do jogo é salvar Harry Potter e Draco Malfoy da Malfoy Manor

↬ Se Draco Malfoy cair no líquido vermelho, ele morre, assim como Harry Potter morre ao cair no líquido azul

Heurísticas de Nielsen

❥ O que são as Heurísticas de Nielsen?

↬ As Heurísticas de Nielsen são um conjunto de 10 princípios heurísticos para design de interface de usuário, criados por Jakob Nielsen.

↬ Essas heurísticas fornecem um conjunto de diretrizes para avaliar e melhorar a experiência do usuário em uma ampla variedade de sistemas interativos.

↬ São amplamente utilizadas no design de interfaces de usuário em todo o mundo para criar interfaces mais intuitivas, eficientes e satisfatórias para os usuários.

↬ Assim, após testes realizados por usuários, algumas mudanças foram realizadas afim de alavancar a funcionalidade do site e a facilitar seu uso.
